# Dataset

This folder contains the custom dataset used to train the Squire Natural Language Understanding (NLU) model.

The dataset was built for productivity-related requests and combines manually created examples with samples adapted from public NLU benchmark datasets, including:

- ATIS
    
- SNIPS
    
- TOPv2
    
- BANK77
    

These samples were rewritten, extended, and annotated to support the productivity scenarios used in Squire, such as tasks, meetings, notes, and progress management.
